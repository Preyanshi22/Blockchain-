# ArbiLearn — Web3 Education Platform ⚡

![ArbiLearn Banner](https://raw.githubusercontent.com/ArbitrumFoundation/docs/HEAD/static/img/arbitrum-logo-dark.svg)

> **Arbitrum Builder Labs — LamprosDAO Submission**

ArbiLearn is a fully responsive, premium Web3 education platform designed conceptually precisely to teach new developers the fundamentals of Layer 2 scaling, blockchain principles, and core Web3 concepts through beautiful interactive experiences. Now modernized with **React, Vite, and React Router**.

## 🚀 Key Features

1. **Home** — Why Ethereum needs Layer 2, explaining Optimistic Rollups.
2. **Web3 Concepts** — Side-by-side breakdowns of technical deep dives and FAQs.
3. **Live Prices** — Real-time fetching of asset prices via CoinGecko API with filters and portfolio tracking.
4. **Block Simulator** — An interactive environment testing SHA-256 hashes, Nonces, and cascade consequences breaking immutability.

## 🛠️ Tech Stack & Modernization

- **React 18**: Modern component-based architecture.
- **Vite**: Ultra-fast build tool and development server.
- **React Router**: Seamless single-page navigation.
- **CSS3 (Vanilla)**: Preserved custom Glassmorphism CSS engine.
- **Web Crypto API**: SHA-256 hashing execution powering the Simulator logic.

## 🏃 Running Locally

1. Clone or download this repository.
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Build for production: `npm run build`
