How It Works

What is SHA-256?

A cryptographic hashing algorithm that takes any input data and generates a fixed-size 64-character hexadecimal string. Changing even one letter changes the entire output drastically.

What is a Nonce?

A "Number used ONCE". In mining, it's the arbitrary number miners increment repeatedly to try and find a valid hash that starts with enough leading zeroes.

Why does the chain break?

Block 2 includes Block 1's Hash as part of its own calculation. If you alter Block 1, its hash changes. That change propagates immediately to Block 2's calculation, causing a mismatch and breaking the chain recursively.
