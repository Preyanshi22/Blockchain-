🎯 What is a Nonce?

A nonce is a number miners increment until the block's hash meets the difficulty requirement. It's the 'trial and error' of proof-of-work.

🔐 What is SHA-256?

SHA-256 converts any input into a fixed 64-character output. Change even one character of input and the entire output changes completely.

⛓️ Why Does the Chain Break?

Each block stores the previous block's hash. Changing Block 1 changes its hash, which no longer matches what Block 2 stored — breaking the link.
